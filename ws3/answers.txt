Part B — File Permissions

B1: id    # uid = user ID (the individual account); gid = primary group ID. uid=1039(ishigaki-cs6) gid=1039(ishigaki-cs6) groups: ishigaki-cs6, video, docker, render
B2: ls -l fruits.txt    # -rw-r-----  owner(rw-)=read+write, group(r--)=read only, other(---)=no access
B3: umask    # 0002 -> new files get 666 & ~002 = 664 (-rw-rw-r--); verified with: touch umask_test.txt && ls -l umask_test.txt
B4: chmod 000 secret.txt    # cat secret.txt -> "cat: secret.txt: Permission denied"
B4: chmod u+rw secret.txt    # restore owner read+write with symbolic notation
B5: chmod 640 fruits.txt    # octal value = 640 (owner rw, group r, other none)
B6: find ~/cs131/ws3 -type f -user "$(whoami)" -perm -u+w
