# CS 131 — Midterm 1 Review

Interactive study tools and printable cheat sheets. The `.html` files run in any
browser with no internet needed; the `.pdf` sheets are made to print.

## Interactive (open in a browser)
- **cs131_midterm1_study_app.html** — concept cheat sheets, the grep & regex lab
  (mental model, live tester, predict-the-match drills, matching game), and a
  63-question quiz with instant feedback.
- **cs131_ws1_practice.html** — ws1 trainer: Linux basics + grep/regex (28 tasks).
- **cs131_ws2_practice.html** — ws2 trainer: the git workflow (16 tasks).
- **cs131_ws3_practice.html** — ws3 trainer: redirection, permissions, jobs (24 tasks).

Each task: type your command, **Check** it (lenient match), **Reveal** the model
answer + explanation, and self-grade. Progress is tracked per session.

## Printable cheat sheets (PDF, 2 pages each)
- **cs131_concepts_grep_cheatsheet.pdf** — concept summaries (slides 01–05 + Git)
  and the grep lab, in a clean layout.
- **cs131_midterm1_cheatsheet.pdf** — command + worked worksheet-solution reference.

## Use on a Mac
```
git clone https://github.com/kgr-17/cs131.git
cd cs131/midterm1-review
open cs131_midterm1_study_app.html      # opens in your default browser
```
